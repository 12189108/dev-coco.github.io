创建 Telegram 机器人

2022-02-08 / 1 min read #其它

创建 Telegram 机器人，获取 token。

在 Telegram 搜索 @BotFather，如果是第一次使用，需要在对话窗口的下方点击开始。

输入命令/newbot，然后输入一个机器人的名字。

再设置一个机器人的用户名，必须以 bot 作为结尾。

注册成功后会发送机器人的 token。
