dmesg

2022-02-04 / 2 min read #Linux命令

用于显示开机信息。

kernel 会将开机信息存储在 ring buffer 中。您若是开机时来不及查看信息，可利用 dmesg 来查看。开机信息亦保存在 /var/log 目录中，名称为 dmesg 的文件里。

语法

dmesg [-cn][-s <缓冲区大小>]

参数说明：

-c 　显示信息后，清除 ring buffer 中的内容。
-s<缓冲区大小> 　预设置为 8196，刚好等于 ring buffer 的大小。
-n 　设置记录信息的层级。
