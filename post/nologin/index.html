nologin

2022-02-02 / 2 min read #Linux命令

拒绝用户登录系统。

补充说明

nologin 命令可以实现礼貌地拒绝用户登录系统，同时给出信息。如果尝试以这类用户登录，就在 log 里添加记录，然后在终端输出 This account is currently not available 信息，就是这样。一般设置这样的帐号是给启动服务的账号所用的，这只是让服务启动起来，但是不能登录系统。

语法

nologin

实例

Linux 禁止用户登录：
禁止用户登录后，用户不能登录系统，但可以登录 ftp、SAMBA 等。我们在 Linux 下做系统维护的时候，希望个别用户或者所有用户不能登录系统，保证系统在维护期间正常运行。这个时候我们就要禁止用户登录。
1、禁止个别用户登录，比如禁止 lynn 用户登录。

passwd -l lynn

这就话的意思是锁定 lynn 用户，这样该用户就不能登录了。

passwd -u lynn

上面是对锁定的用户 lynn 进行解锁，用户可登录了。
2、我们通过修改/etc/passwd文件中用户登录的 shell

vi /etc/passwd

更改为：

lynn:x:500:500::/home/lynn:/sbin/nologin

该用户就无法登录了。
3、禁止所有用户登录。

touch /etc/nologin

除 root 以外的用户不能登录了。
