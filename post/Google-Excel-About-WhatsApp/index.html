Google Excel WhatsApp 相关

2022-06-14 / 1 min read #WhatsApp #Excel

使用 Google Excel 函数获取 WhatsApp 群组的名称和头像。

获取群组名称

=INDEX(IMPORTXML(A1, "//h3"), 2)

原理分析

先分析网页源码，定位群组名称使用的 HTML 标签和位置。在第二个 h3 标签显示的是群组的名称。

使用 IMPORTXML 函数，用 XPath 语法 //h3 获取网页的 h3 标题。

因为预期的结果是第二个，需要使用 INDEX 函数，获取第二行的内容。

获取群组头像

=IMAGE(INDEX(IMPORTXML(A1, "//img/@src"), 3))

原理分析

使用 IMPORTXML 函数，用 XPath 语法 //img/@src 获取网页所有 img 标签的 src 属性。

第三行的链接是群组的头像，需要使用 INDEX 函数，获取第三行的内容。

最后再使用 IMAGE 函数显示图片。
