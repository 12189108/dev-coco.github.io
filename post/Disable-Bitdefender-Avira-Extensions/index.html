屏蔽 Bitdefender 和 Avira 安装插件

2022-08-18

屏蔽 Bitdefender 和 Avira 杀毒软件强制给浏览器安装插件。

有的杀毒软件会强制给浏览器安装一些“安全性”插件，而且每次新建浏览器人员的时候都会被强制安装上。

即使卸载软件的时候，官方提供的卸载工具也不会完全将这些插件完全卸载掉，还是会被强制安装到浏览器。那么就可以把这些插件的 ID 设置到浏览器的插件黑名单内，这样就可以彻底屏蔽掉，不会再强制安装浏览器了。
下载链接：Disable-Bitdefender-and-Avira-Extensions

同时支持 Chrome、Brave、Edge 三款浏览器，安装后重启浏览器即可生效。
