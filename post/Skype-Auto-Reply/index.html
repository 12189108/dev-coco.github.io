Skype 自动回复

2022-12-08

Skype 自动回复用户自定义消息。

下载链接：Chrome 插件合集

作用

由于 Skype 社交软件原生没有自动回复等便捷功能，当咨询量大，消息多或者离开的时候，往往会出现回复不及时的情况，那么自动回复就能很好的解决这个问题。

设置

安装插件后，点击右键，选项，进行设置。

域名前缀

登陆 Skype 网页版后，在浏览器的控制台找到网络，在请求头里面的请求链接找到域名前缀。

然后在插件设置里选择对应的域名前缀。在 Skype 网页版没有刷新的情况下，一般情况域名前缀不会变，如果刷新页面，可能域名前缀会发生变化。

Skype Token

登陆 Skype 网页版后，在浏览器的控制台找到网络，在请求头里面找到skypetoken，等号(=)后面的就是Skype Token。

循环时间

以秒为单位，每次循环扫描界面的间隔时间。例如设置 30，那么每 30 秒就会循环扫描一次界面，检测是否有新消息。修改后不会实时生效，需要刷新 Skype 网页版后才能生效。

音量

默认在发送消息之前会发出 🔔叮～ 的声音，可以自行调节音量声音的大小。

需要回复的内容

填写需要自动回复的内容，必须包含「自动回复」。

使用方法

  1. 打开 Skype 网页版后，需要在左侧的聊天列表显示出最新聊天的联系人。因为程序只会扫描当前界面显示出来的联系人，如果当前界面不显示最新发消息的联系人，那么此功能就无法正常使用。
  2. 点击插件后，会在左上角显示开始按钮，点击按钮后即可按照设置的参数自动运行。
  3. 开始运行后不能关闭浏览器，进程也不能被冻结，否则会无法正常运行。
