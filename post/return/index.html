return

2022-02-01

从函数中退出并返回数值。。

概要

return [n]

主要用途

使得 shell 函数退出并返回数值，如果没有指定 n 的值，则默认为函数最后一条命令执行的返回状态。

参数

n（可选）：整数。

返回值

返回值为你指定的参数 n 的值，如果你指定的参数大于 255 或小于 0，那么会通过加或减 256 的方式使得返回值总是处于 0 到 255 之间。
在函数外执行 return 语句会返回失败。

例子

<span class="token shebang important">#!/usr/bin/env bash# 定义一个返回值大于 255 的函数 <span class="token function-name function">example() {  return 259}# 执行函数 example# 显示 3echo $?

注意

该命令是 bash 内建命令，相关的帮助信息请查看help命令。
