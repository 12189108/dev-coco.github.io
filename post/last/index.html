last

2022-02-03 / 2 min read #Linux命令

列出目前与过去登入系统的用户相关信息。

补充说明

last 命令用于显示用户最近登录信息。单独执行 last 命令，它会读取/var/log/wtmp的文件，并把该给文件的内容记录的登入系统的用户名单全部显示出来。

语法

last (选项) (参数)

选项

-a：把从何处登入系统的主机名称或 ip 地址，显示在最后一行；
-d：将 IP 地址转换成主机名称；
-f <记录文件>：指定记录文件。-n <显示列数>或-<显示列数>：设置列出名单的显示列数；
-R：不显示登入系统的主机名称或 IP 地址；
-x：显示系统关机，重新开机，以及执行等级的改变等信息。

参数

  • 用户名：显示用户登录列表；

  • 终端：显示从指定终端的登录列表。

实例

last 命令用了显示用户登录情况，以下是直接显示固定行数的记录：
