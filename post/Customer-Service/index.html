自助问答客服

2022-04-06

自助问答客服是一个根据关键词快速回复自定义内容的网页。

特点：

  1. 根据关键词回复自定义内容。
  2. 关键词搜索。
  3. Telegram 接收 Google 表单回复。
  4. 自适应页面。

源码：https://github.com/dev-coco/Customer-Service

配置

Telegraam 接收 Google 表单回复

在表单右上角三个点的位置，选择脚本编辑器。

放入 TelegramScript.js 文件内的代码。
在 TelegramBotToken 放入 Telegram 机器人的 token。
myID 放入自己账号的 ID，在 Telegram 搜索 @userinfobot 就可以查询到账号的 ID。
全部设置好后，在左侧选择触发器。

在右下角点击添加触发器。

选择活动类型这里选择表单提交时，然后保存。

如果是第一次使用的话会需要账号授权。选择邮箱账号。

点击左侧的高级。

点击转至项目。

滑倒底部，点击允许。

设置完成后，当提交表单的时候会在 Telegram 发消息回复表单的内容。

保存表单的 ID，留着待会使用。

表格

创建一个表格，表格的权限要设置成查看权限。
在表格的 A 列输入关键词。
在表格的 B 列输入回复的内容。B 列的内容可以使用 HTML 标签。

需要需要换行，无需添加 <br>，在单元格内直接换行即可。
如果在 B 列放入可点击的关键词，需要在 class 属性添加 k，例如：

<span class="k">可点击关键词</span>

如果对 HTML 不太了解，可以使用 富文本编辑器，编辑好内容后点击右上角的 Source 获取源码在粘贴到单元格中。

Customer-Service.html

第 14 行的 iframe 中，FormID 替换成 Google 表单的 ID。

在 msg-text 这个 class 属性下面可以自定义初始内容。

sheetID 放入加密后的 Google 表格 ID
apiKey 放入加密后的 Google API 密钥。
使用 编码转换工具 加密，选择字符加密，放入内容，点击转换按钮，在右侧就会输出加密后的内容。

全部配置好后，上传到服务器就可以使用了。
