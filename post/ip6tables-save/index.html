ip6tables-save

2022-02-01

保存 ip6tables 表配置。

补充说明

ip6tables-save 命令将 Linux 内核中 ip6tables 表导出到标准输出设备上。

语法

ip6tables-save (选项)

选项

-c：指定在保存iptables表时，保存当前的数据包计数器和字节计数器值；
-t：指定要保存的表的名称。
