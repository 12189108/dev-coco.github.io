人脸识别 API 接口

2022-10-26 / 2 min read #Javascript

简单易用的人脸识别 API 接口，可识别年龄、性别、人种，完全免费。

基本信息

请求类型: HTTP
接口地址: https://script.google.com/macros/s/AKfycbzgobSFBW0sbiKRrLIYYjKDbSUPEJgDpZwv-L0Gsm1bv6_pBeYeCWVQvvNILM-TvD9i/dev
请求方式: GET
数据类型: JSON

接口描述

通过图片链接人脸识别年龄、性别、人种。

请求参数

参数名称 是否必须 类型 描述
type true string 识别类型，支持同时使用多种，例：age,race,gender
imageUrl true string 图片链接

例子：

const param = {
  type: 'race,age',
  imageUrl: 'https://example.com/test.jpg'
}
const json = await fetch(`https://script.google.com/macros/s/AKfycbzgobSFBW0sbiKRrLIYYjKDbSUPEJgDpZwv-L0Gsm1bv6_pBeYeCWVQvvNILM-TvD9i/dev?${new URLSearchParams(param)}`).then(response => response.json())

响应数据

参数名称 是否必须 类型 描述
age false string 年龄
race false string 人种
gender false string 性别

例子：

{
  "age": 38,
  "race": "white",
  "gender": "Man"
}

响应错误代码

错误代码 描述
Error: 201 图片链接错误
Error: 202 类型错误
Error: 203 类型数量错误
Error: 204 服务器出错
