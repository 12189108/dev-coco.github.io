61.1 Lesson

ES6 Classes

Question:

Classes in ES6
Fill in the blanks to declare a class Point with a constructor initializing its x and y members.

Answer:

class Point {
  constructor(a, b) {
    this.x = a;
    this.y = b;
  }
  getX() { return this.x; }
  getY() { return this.y; }
}

Question:

Class Methods in ES6
Fill in the blanks to output "Rex barks."

Answer:

class Dog {
	constructor(name) {
		this.name = name;
	}
	bark() {
		console.log(this.name + ' barks.');
	}
}
let d = new Dog('Rex');
d.bark();

Question:

Inheritance in ES6
Fill in the blanks to declare a class Student which inherits from the Human class.

Answer:

class Human {
	constructor(name) {
		this.name = name;
	}
}
class Student extends Human {
	constructor(name, age) {
		super(name);
		this.age = age;
	}
}
