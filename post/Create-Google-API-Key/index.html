申请 Google API 密钥

2022-04-05 / 1 min read #其它

申请 Google API 密钥的方法。

申请链接：https://console.cloud.google.com/apis/
在上方点击选择项目，如果之前有创建过，点击项目的名称。

点击右上角的新建项目。

自定义项目名称，点击创建。

点击左侧的凭据。

在上方点击创建凭据，选择 API 密钥。

这样就申请好 Google 的API 密钥了。
