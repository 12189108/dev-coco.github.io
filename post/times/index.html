times

2022-02-01

显示进程累计时间。

主要用途

打印出 shell 及其子进程累计使用的用户时间和系统时间。

返回值

总是返回成功。

例子

# 执行命令
times
# 返回结果
0m0.037s 0m0.009s
0m0.010s 0m0.024s
# 根据times(2)的man手册，对应关系如下：
# 用户时间        | 系统时间
# 子进程的用户时间 | 子进程的系统时间

注意

  • 该命令是 bash 内建命令，相关的帮助信息请查看help命令。
