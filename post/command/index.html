command

2022-01-31

调用并执行指定的命令。

补充说明

command 命令调用指定的指令并执行，命令执行时不查询shell函数。command 命令只能够执行 shell 内部的命令。

语法

command (参数)

参数

指令：需要调用的指令及参数。

实例

使用 command 命令调用执行echo Linux，输入如下命令：

command echo Linux            #调用执行shell内部指令

上面的命令执行后，将调用执行命令echo Linux
