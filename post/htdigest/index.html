htdigest

2022-02-01

Apache 服务器内置工具。

补充说明

htdigest 命令是 Apache 的 Web 服务器内置工具，用于创建和更新储存用户名、域和用于摘要认证的密码文件。

语法

htdigest (选项) (参数)

选项

-c：创建密码文件。

参数

  • 密码文件：指定要创建或更新的密码文件；
  • 域：指定用户名所属的域；
  • 用户名：要创建或者更新的用户名。
