pwck

· 2022-02-02 · # Linux命令

用来验证系统认证文件内容和格式的完整性。

补充说明

pwck 命令用来验证系统认证文件/etc/passwd和/etc/shadow的内容和格式的完整性。

语法

pwck (选项) (参数)

选项

-q：仅报告错误信息；
-s：以用户 id 排序文件“/etc/passwd”和“/etc/shadow”;-r：只读方式运行指令。

参数

  • 密码文件：指定密码文件的路径；

  • 影子文件：指定影子文件的路径。

实例

pwck /etc/passwd
